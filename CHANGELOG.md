# 更新日志

本文档用于记录项目的重要变更。

## 2026-06-29

### 新增

- 新增集成自动化模块，支持应用级自动化列表、自动化编辑页与工作流保存。
- 新增自动化触发器节点、查询节点、新增数据节点、更新数据节点、删除数据节点、HTTP 请求节点、条件分支节点。
- 新增自动化版本快照能力，支持版本列表查询与历史版本恢复。
- 新增自动化结构化持久化表：
  - `automation_flows`
  - `automation_flow_versions`
  - `automation_nodes`
  - `automation_edges`
- 新增自动化运行日志能力，支持查看流程运行记录、节点执行记录、输入输出载荷、耗时与重试来源。
- 新增失败处理能力：
  - 整个流程重头触发
  - 错误节点断点重试
- 新增表单记录运行时编辑、删除接口，并接入自动化触发链路。
- 新增前端 Next.js 代理接口，用于转发自动化详情、版本、运行日志与重试相关请求。

### 调整

- 将自动化编辑器重构为更适合复杂节点配置的暗色工作区布局，属性面板仅在选中节点时显示，最大宽度限制为 `500px`。
- 将工作流名称、说明、启停状态与版本管理上移到编辑器左上角头部区域，移除触发器节点中的工作流级配置项。
- 调整自动化画布交互，点击空白区域时隐藏属性配置，减少右侧配置区常驻造成的干扰。
- 调整条件分支节点的数据源策略，改为可使用全部上游节点产生的数据，而不是仅使用直接上级节点。
- 调整条件分支配置模型，支持：
  - 所有数据均可通过
  - 按条件规则进入
  - 按公式定义进入
- 调整触发器节点配置逻辑：
  - 初始节点名称、说明不可编辑
  - 变更字段仅在编辑前/编辑后事件下展示
- 调整应用内导航，在集成自动化页面隐藏不必要的表单导航干扰项。
- 更新 OpenAPI 描述与前端生成客户端，补齐本轮自动化相关接口的类型定义。

### 修复

- 修复非触发器节点错误显示工作流配置的问题。
- 修复节点属性抽屉在选择部分配置值后意外关闭的问题。
- 修复拖拽连线到空白画布后未弹出新增节点对话框的问题。
- 修复自动化编辑器布局撑高页面、导致浏览器出现多余滚动条的问题。
- 修复运行日志接口在前后端版本不一致时返回空响应，前端表现为 `503` 的问题；现在会返回明确的重启后端提示。
- 修复自动化运行日志查看路径 `404` 排查链路，确认问题来源为旧版后端进程未重启。
- 修复新增字段配置中公式输入框、多条数据来源节点映射等错误交互。
- 修复触发器节点以外的节点仍可编辑不该暴露字段的问题。

### 说明

- 当前自动化运行已支持触发事件：
  - `before_create`
  - `after_create`
  - `before_update`
  - `after_update`
  - `before_delete`
  - `after_delete`
- 当前自动化执行已支持节点类型：
  - 获取单条数据
  - 获取多条数据
  - 新增数据
  - 更新数据
  - 删除数据
  - HTTP 请求
- 当前错误节点重试已经从“整条流程重新执行”调整为“基于失败节点输入快照的断点恢复执行”。
- 若本地仍访问不到运行日志接口，需要确保后端已在本轮代码更新后重新启动。

### 后续优化

- 为自动化运行日志增加后端分页、筛选、统计汇总与日志保留策略。
- 为 HTTP 请求节点补充更完整的响应状态、响应头与错误上下文展示。
- 统一自动化接口到前端生成 API Client，减少运行日志相关页面对手写 `fetch` 的依赖。
- 为自动化执行链路增加事件来源标记、递归深度限制与循环触发防护，避免未来出现自动化自触发套娃风险。
- 补齐更完整的条件表达式编辑器、字段映射编辑器与运行前校验能力。

### 文档

- 删除重复职责的 `README.log`，统一由 `README.md` 承载项目说明，由 `CHANGELOG.md` 承载版本更新记录。
- 更新 `README.md`，补充集成自动化、运行日志、自动化存储结构、本地运行说明与下一步规划。

## 2026-06-26

### 新增

- 新增应用管理接口与前端应用列表页面。
- 新增空白应用创建能力。
- 新增应用卡片工具栏能力，包括访问、启用/关闭、编辑名称、设置入口、删除。
- 新增应用内置系统页面导航。
- 新增应用下表单导航与分组导航。
- 新增导航分组递归结构支持。
- 新增导航拖拽排序能力。
- 新增表单设计器页面。

  - `didMount(ctx)`
  - `onFieldEvent(ctx)`

- 新增表单预览能力。
- 新增表单 Schema 草稿保存能力。
- 新增表单发布能力。
- 新增表单历史版本查询与恢复能力。
- 新增表单运行时页面。
- 新增表单记录保存与查询能力。
- 新增基于 `@hey-api/openapi-ts` 的前端 API Client 自动生成能力。
